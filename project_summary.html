Pixmos - 비회원 중심 픽셀 드로잉 커뮤니티 MVP

🎨 "여러 사용자가 점을 찍듯 자유롭게 그림을 공유하는 비회원 중심 플랫폼"

📋 프로젝트 개요

Pixmos는 회원가입 없이 누구나 쉽게 픽셀 아트를 그리고 공유할 수 있는 비회원 중심 커뮤니티 서비스입니다. Windows 95 스타일의 복고풍 픽셀 감성 UI로 모바일 최적화된 드로잉 경험을 제공합니다.

🎨 주요 기능 (비회원 기준)

1. 픽셀 드로잉 도구 및 그리기 화면

  • 툴바:
    - Undo(되돌리기), Redo(다시 실행)
    - 지우개, 펜(기본 선택), 페인트(연속된 영역 채우기)
    - 컬러 팔레트: 12개 색상, 최근 사용 색상 순서대로 자동 배치
    - 색 지정: 팔레트에서 선택한 색상은 왼쪽부터 차례로 쌓임
  • 캔버스:
    - 24x24 격자, 최대 가로폭 390px(모바일 기준), 각 셀은 정사각형
    - 실시간 미리보기, 터치/드래그로 그리기 지원
  • 저장 팝업(바텀시트):
    - 저장 시 하단에 팝업 등장, 저장/취소 등 액션 제공

2. 홈 피드(커뮤니티) 기능

  • 홈 피드: 모든 사용자의 그림(썸네일), 닉네임, 글, 날짜를 한눈에 볼 수 있는 메인 피면
  • 좋아요(스크랩): 마음에 드는 작품을 저장/스크랩
  • 댓글: 각 그림별로 댓글 작성 및 소통
  • 신고: 부적절한 콘텐츠 신고
  • 정렬/필터: 최신순, 인기순 등 정렬 지원(예정)
  • 간편 게시: 닉네임, 글, 비밀번호만 입력하면 게시
  • 비밀번호 보안: 비밀번호로 수정/삭제 권한 관리
  • 썸네일 자동 생성: 드로잉 결과물 자동 썸네일화

🛠️ 기술 스택

HTML5 CSS3 JavaScript (Vanilla) Canvas API IndexedDB PWA Responsive Design

🎨 UI/UX 컨셉

복고풍 픽셀 감성 (Windows 95 스타일) + 모바일 최적화

  • 툴바: 상단에 아이콘 버튼(undo, redo, 지우개, 펜, 페인트, 컬러팔레트, 색 지정) 일렬 배치
  • 컬러 팔레트: 12개, 최근 사용 색상은 왼쪽부터 순차적으로 쌓임
  • 캔버스: 24x24 격자, 최대 390px, 명확한 픽셀 경계선
  • 저장 팝업(바텀시트): 저장 시 하단에 등장, 직관적 액션 제공
  • 홈 피드: 카드형 썸네일, 닉네임, 글, 날짜, 좋아요, 댓글, 신고 버튼 등 커뮤니티형 UI
  • 폰트: 픽셀 폰트 또는 시스템 폰트
  • 버튼 스타일: 입체감 있는 3D 버튼 효과
  • 반응형: 모바일 기준, 터치/드래그 지원

💡 서비스 핵심 가치

Pixmos는 "점을 찍듯 자유롭게"라는 컨셉으로, 누구나 쉽게 접근할 수 있는 픽셀 아트 커뮤니티를 지향합니다. 회원가입의 번거로움 없이 즉시 창작을 시작할 수 있고, Windows 95의 복고풍 감성으로 따뜻한 커뮤니티 분위기를 조성합니다.
